# 🌿 ระบบร้านค้าออนไลน์สำหรับจำหน่ายสกินแคร์ (Skincare E-Commerce System)

## รายละเอียดโครงงาน

โครงงานนี้เป็นระบบร้านค้าออนไลน์สำหรับจำหน่ายผลิตภัณฑ์สกินแคร์ โดยพัฒนาขึ้นเพื่อให้ผู้ใช้งานสามารถเลือกซื้อสินค้า สั่งซื้อ และติดตามคำสั่งซื้อผ่านเว็บไซต์ได้อย่างสะดวก พร้อมทั้งมีระบบจัดการข้อมูลสำหรับผู้ดูแลระบบ (Admin) เพื่อจัดการสินค้า หมวดหมู่ และคำสั่งซื้อ

---

## วัตถุประสงค์ของโครงงาน

* พัฒนาระบบร้านค้าออนไลน์สำหรับจำหน่ายสกินแคร์
* ศึกษาและประยุกต์ใช้ Git และ GitHub ในการจัดการเวอร์ชันของโปรแกรม
* ศึกษาการออกแบบสถาปัตยกรรมซอฟต์แวร์และฐานข้อมูล
* ฝึกการใช้งาน SourceTree ในการจัดการโครงการร่วมกับ GitHub

---

## ความสามารถของระบบ

### สำหรับลูกค้า

* สมัครสมาชิก
* เข้าสู่ระบบ
* แสดงรายการสินค้า
* ค้นหาสินค้า
* เลือกสินค้าตามหมวดหมู่
* เพิ่มสินค้าในตะกร้า
* สั่งซื้อสินค้า
* ตรวจสอบประวัติการสั่งซื้อ

### สำหรับผู้ดูแลระบบ

* จัดการข้อมูลสินค้า
* จัดการหมวดหมู่สินค้า
* จัดการคำสั่งซื้อ
* จัดการข้อมูลสมาชิก
* อัปเดตสถานะการสั่งซื้อ
